# Express_basic_test
render a simple page by express-handlebars


## 使用方法
1. 確認使用環境已有node.js及npm
2. 下載(clone)此專案
    ```
    git clone https://github.com/gyurt32413/Express_basic_test
    ```
3. 透過終端機安裝npm套件
    ```
    npm install
    ```
4. 透過終端機安裝nodemon
    ```
    npm i -g nodemon
    ```
5. 透過nodemon執行該專案
    ```
    nodemon app.js
    ```
6. 於瀏覽器輸入`http://localhost:3000/` 即可進入該網頁


## 開發工具
+ Node.js 14.16.0
+ Express 4.17.2
+ Express-Handlebars 5.3.3
+ Bootstrap 4.3.1
+ nodemon 2.0.15
